# DASH-Training-VSCode

Repository for the training of new users on VS Code on Posit Workbench. Includes GitHub workflow integration, Posit Workbench setup instructions, Python package management using Pixi, and Databricks integration via databricks-connect and Databricks Extension.

Further learning resources for VS Code can be found on the DASH Playbook [here](https://dash-connect-prd.azure.defra.cloud/DASH-Playbook/docs/workbench.html#vs-code-on-workbench) & [here](https://dash-connect-prd.azure.defra.cloud/DASH-Playbook/docs/vs_code.html#vs-code-databricks)
